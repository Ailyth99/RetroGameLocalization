AirouPuzzleTool: 
Toolkit for PSP Airou de Puzzle ARC/PAK & ZC archives.

Unpack: 
AirouPuzzleTool.exe -u <input.arc> [-o out_dir] (Auto ZC-extraction).
Repack: 
AirouPuzzleTool.exe -r <folder> [-o out.arc] (Auto ZC-compression).

Fonts and script texts are contained in SYSTEM.PAK.

-------------------
Import And Export Script Text

AirouPuzzle_text_tool：
Export: text_tool -export <original.bin>
Import: text_tool -import <txt> <bin> <char_tbl.csv>
Count:  text_tool -count <txt>
