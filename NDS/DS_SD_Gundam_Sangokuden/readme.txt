
A minimal CLI tool for exporting and importing text from SD Gundam Sangokuden Brave Battle Warriors: Shin Mirisha Taisen (NDS).

Usage

1. Export file/folder:  tool [-ruby] <.dat path>
2. Count unique chars:  tool -count <txt_folder>
3. Import translation:  tool -import <text.txt> <original.dat> [output.dat]
